# win-acme MSI Installer

MSI package for **win-acme (WACS)**.
Built automatically via GitHub Actions.

Company: **Ascensio System SIA**

---

## Installation

Installs to:

```
C:\Program Files\Win-Acme\
```

During installation it:

* Extracts the official win-acme release
* Adds the install directory to the system **PATH**
